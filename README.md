# :rocket: React Hooks

## :exclamation: Requirements

- Docker
- Node >= 10
- Yarn

## :computer: Getting started

### Development mode

- `yarn`: Installs workspace dependencies
- `yarn start`: Starts the project
- Open the webapp at `http://localhost:8081`

### Stopping

- `yarn stop`: Stops the project
